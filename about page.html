Why This Website Was Created

BMW M3 Legacy was developed as a comprehensive and educational website project by a tertiary-level student in the BSc Applied Business Computing programme. This site was created for the Web Development module to demonstrate both technical web design skills and a deep appreciation for automotive innovation.

The core purpose of this website is to explore and present the legacy of the BMW M3 — a vehicle line renowned for its connection to motorsport, exceptional engineering, and influence on performance vehicle culture. By combining text, visuals, and interactive elements such as slideshows and a feedback form, the site aims to offer a meaningful experience for visitors of various backgrounds — whether they are automotive enthusiasts, casual browsers, or students conducting research.

The BMW M3 has evolved through six major generations since its introduction in 1986, and each one has played a significant role in shaping the expectations for what a performance sedan should be. Through this website, users can learn about each generation's unique characteristics, special editions, and technical innovations. The aim is not only to educate but also to inspire interest in automotive history, design, and technology.

Website Goals:

  • Provide a structured and accessible archive of all BMW M3 generations
  • Use modern web technologies to build a clean, responsive, and visually engaging website
  • Promote user engagement through a functional and accessible feedback form
  • Develop a site that aligns with W3C standards and showcases professional web design practices

Technologies and Tools Used:

  • HTML5 – for semantic and structured page content
  • CSS3 – for responsive design and visual styling
  • JavaScript – to create interactive components like image slideshows and lightboxes
  • Formspree – to handle contact form submissions without a backend
  • GitHub Pages – for version control and public hosting
